1，MySQL数据库导入与导出

导出现有数据库数据：
mysqldump -u用户名 -p密码 数据库名称 >导出文件路径 # 结构+数据
mysqldump -u用户名 -p密码 -d 数据库名称 >导出文件路径 # 结构
导入现有数据库数据：
mysqldump -uroot -p密码 数据库名称 < 文件路径
