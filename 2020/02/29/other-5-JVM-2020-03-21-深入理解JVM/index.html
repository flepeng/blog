如何判断对象可以回收

1、引用计数法

互相引用循环的问题无法回收

2、可达性分析

我们知道Java 进行GC的时候会从GC root进行可达性判断，常见的GC Root有如下：

  1. 通过System Class Loader或者Boot Class Loader加载的class对象，通过自定义类加载器加载的class不一定是GC Root
  2. 处于激活状态的线程
  3. 栈中的对象
  4. JNI栈中的对象
  5. JNI中的全局对象
  6. 正在被用于同步的各种锁对象
  7. JVM自身持有的对象，比如系统类加载器等。

在调查内存泄漏原因的时候可以根据GC Root来推导
