Random产生随机数（掌握）

  • 概述：

    • Random类似Scanner，也是Java提供好的API，内部提供了产生随机数的功能

    • API可以简单理解为Java已经写好的代码

  • 使用步骤：

    1. 导入包

      import java.util.Random;

    2. 创建对象

      Random r = new Random();

    3. 产生随机数

      int num = r.nextInt(10);

      解释： 10代表的是一个范围，如果括号写10，产生的随机数就是0-9，括号写20，参数的随机数则是0-19
