开工交底

交底是装修施工前非常重要的的步骤。

主要目的在于让施工人员、监理人员、设备厂家知晓施工方案，对方案疑 问及各方需求、各方衔接提前沟通处理。

监理会针对施工方案及现场情况给出相关建议。例如：结构墙不能拆改、 拆除前需封堵下水，不然会导致下水堵 塞、拆除断水断电避免安全隐患、原墙面腻子层是否需要铲除等。

墙皮

拆墙皮

红砖结构老房改造

1、如沙灰层空鼓、粉化严重，建议铲除至红砖层 → 水泥抹灰找平 → 石膏找平 → 贴网格布 → 刮腻子处理 → 刷漆
2、如沙灰层空鼓、粉化不严重，可以铲除至沙灰层 → 石膏找平 → 贴网格布(根据使用需求自选) → 刮腻子处理 → 刷漆

钢混结构老房改造

1、如原墙面是耐水腻子，粉化、空鼓不严重的情况下可铲除至耐水腻子层 → 局部石膏找平(以铲除后墙面情况为准) → 贴网格布(根据使用需求自选) → 刮腻子处理 → 刷漆
2、如原墙面是非耐水腻子，建议铲除至原墙面水泥基层 → 石膏找平 → 贴网格布(轻体墙、新建墙、石膏找平过厚的地方) → 刮腻子处理 → 刷漆

新房

1、原墙面如果是非耐水腻子(墙面是腻子层，下边是石有层， 直接腻子层洒水若墙面有渗水现象，用指甲、钥匙等破坏一下，如有脱落就是非耐水腻子)建议铲除至原墙面水泥基层，轻体墙、新建墙、施I洞处需要贴网格布处理。
2、原墙面如果是耐水腻子，无大面积空鼓、粉化等现象可以不铲除，墙面做防裂处理效果最佳，以现场实际情况为准。

外墙内保温

1）内保温墙体需不需要拆除； 如果小区统一新做了外保温，室内温度适宜的情 况下可以考虑拆除； 如果小区没有新做外保温，内保温建议不要破坏， 避免影响取暖效果； 施工方铲除墙面时，提前约定不要破坏内保温 （拆除、新做内保温容易恶意增项）
2）内保温墙体如何处理刷防水涂料，隔绝湿气（建议贴网格布防裂处理） 贴耐水石膏板隔绝湿气（石膏板接 缝处做防裂）

石膏找平

1、披刮山鹅或首和牌底层石膏1至2遍(如需美巢墙坭另加5元/平米)，石膏找平厚度S 小于等于 10mm,如墙体偏差超过10mm,需采取水泥沙浆，采用水泥沙浆找平30元/平米，石膏底层找平标准为顺平，如需做阴阳角水平垂直找直每米另计30元。
2、门窗洞口面积减半计算。

1、墙顶面批刮墙友批刮平整，墙面误差 小于 5mm；
2、≥5mm墙友找平另行收费，每增加5mm厚度，另加20元/m2；
3、面层腻子、涂料另计;
4、施I项目高度≤3米，> 3米加收高空作业费，每增加1米，收取10%，不足1米按1米计算。

审核:正常应包含1-1.5cm石膏找平，超招出厚度单独收费，请知畴，此项后期容易增项。墙面平整度正常2m内误差不超3mn

网格布防裂

  1. 一般轻体墙、新砌墙、石膏找平过厚、施工洞、内保温 等位置建议贴网格布处理，建议以铲除后墙面实际情况 为准。
  2. 新房如果墙面不做铲除，可以考虑是否需要全墙贴网格 布防裂。
  3. 顶面一般不需要贴网格布。

拉毛

未加入墙固的拉毛（偷工减料） ，后期容易粉化。

油漆涂刷

墙面打磨砂纸印及砂眼，刷漆不会被 覆盖，影响美观

刷漆分色线不均匀，影响美观

新建隔断墙

卫生间新建墙必须制作地梁或止水梁，避免防水因墙体接缝出现渗漏。

墙体要有植筋与原始墙面连接，增加牢固度。

墙面挂铁丝网后再抹灰，起到抗裂作用，有些新建墙未挂网，后期干燥后，会将瓷砖拉裂。（有些施工方被 发现未挂网后，在抹灰层上直接挂网。这种做法不会起到应有的作用） 墙面水泥抹灰后要进行洒水养护，防止抹灰层开裂 影响后期面层装饰施工。

墙地砖

墙地砖铺贴

1）薄贴：地砖上墙、玻化砖（吸水率交底）需要使用薄贴 （粘接剂）工艺，签约前建议提前确认选砖方案。
2）墙压地（墙砖压地砖）：墙压地铺贴美观、防水效果相对较好。 湿区 建议使用墙压地铺贴
3）阳角处理：磨边工艺需要产生加工费，美观，但是后 期易开裂 阳角条防磕碰

验收

墙地砖空鼓，根据 补贴方式及空鼓情 况确定整改方案

吊顶

吊顶验收

吊顶开裂，施工 时需做防裂处理。 防裂处理后，北 方天气出现细小 裂纹属正常现象， 需停暖后修补
