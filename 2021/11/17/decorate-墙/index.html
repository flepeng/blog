拆墙皮

红砖结构老房改造

1、如沙灰层空鼓、粉化严重，建议铲除至红砖层 → 水泥抹灰找平 → 石膏找平 → 贴网格布 → 刮腻子处理 → 刷漆
2、如沙灰层空鼓、粉化不严重，可以铲除至沙灰层 → 石膏找平 → 贴网格布(根据使用需求自选) → 刮腻子处理 → 刷漆

钢混结构老房改造

1、如原墙面是耐水腻子，粉化、空鼓不严重的情况下可铲除至耐水腻子层 → 局部石膏找平(以铲除后墙面情况为准) → 贴网格布(根据使用需求自选) → 刮腻子处理 → 刷漆
2、如原墙面是非耐水腻子，建议铲除至原墙面水泥基层 → 石膏找平 → 贴网格布(轻体墙、新建墙、石膏找平过厚的地方) → 刮腻子处理 → 刷漆

新房

1、原墙面如果是非耐水腻子(墙面是腻子层，下边是石有层， 直接腻子层洒水若墙面有渗水现象，用指甲、钥匙等破坏一下，如有脱落就是非耐水腻子)建议铲除至原墙面水泥基层，轻体墙、新建墙、施I洞处需要贴网格布处理。
2、原墙面如果是耐水腻子，无大面积空鼓、粉化等现象可以不铲除，墙面做防裂处理效果最佳，以现场实际情况为准。

外墙内保温

1）内保温墙体需不需要拆除； 如果小区统一新做了外保温，室内温度适宜的情况下可以考虑拆除； 如果小区没有新做外保温，内保温建议不要破坏， 避免影响取暖效果； 施工方铲除墙面时，提前约定不要破坏内保温 （拆除、新做内保温容易恶意增项）
2）内保温墙体如何处理刷防水涂料，隔绝湿气（建议贴网格布防裂处理） 贴耐水石膏板隔绝湿气（石膏板接 缝处做防裂）

石膏找平

1、披刮山鹅或首和牌底层石膏1至2遍(如需美巢墙坭另加5元/平米)，石膏找平厚度S 小于等于 10mm,如墙体偏差超过10mm,需采取水泥沙浆，采用水泥沙浆找平30元/平米，石膏底层找平标准为顺平，如需做阴阳角水平垂直找直每米另计30元。
2、门窗洞口面积减半计算。

1、墙顶面批刮墙友批刮平整，墙面误差 小于 5mm；
2、≥5mm墙友找平另行收费，每增加5mm厚度，另加20元/m2；
3、面层腻子、涂料另计;
4、施I项目高度≤3米，> 3米加收高空作业费，每增加1米，收取10%，不足1米按1米计算。

审核:正常应包含1-1.5cm石膏找平，超招出厚度单独收费，请知畴，此项后期容易增项。墙面平整度正常2m内误差不超3mn

网格布防裂

  1. 一般轻体墙、新砌墙、石膏找平过厚、施工洞、内保温 等位置建议贴网格布处理，建议以铲除后墙面实际情况 为准。
  2. 新房如果墙面不做铲除，可以考虑是否需要全墙贴网格 布防裂。
  3. 顶面一般不需要贴网格布。

拉毛

未加入墙固的拉毛（偷工减料） ，后期容易粉化。

油漆涂刷

墙面打磨砂纸印及砂眼，刷漆不会被 覆盖，影响美观

刷漆分色线不均匀，影响美观

新建隔断墙

卫生间新建墙必须制作地梁或止水梁，避免防水因墙体接缝出现渗漏。

墙体要有植筋与原始墙面连接，增加牢固度。

墙面挂铁丝网后再抹灰，起到抗裂作用，有些新建墙未挂网，后期干燥后，会将瓷砖拉裂。（有些施工方被 发现未挂网后，在抹灰层上直接挂网。这种做法不会起到应有的作用） 墙面水泥抹灰后要进行洒水养护，防止抹灰层开裂 影响后期面层装饰施工。

  • 家里有孩子，油漆用多好的都不为过——不要轻信所谓大牌，打广告的牌子不一定是大牌，甲醛含量未必少。那些国际品牌贵是贵，但是环保性确实好，健康比什么都重要。

  • 不管你家的墙刷的多好多平整，面漆和底漆一定要一起上。再好的面漆也要刷底漆！不然水泥里的碱性会腐蚀面漆。

  • 毛玻璃背面千万保护好别沾上油漆，很难清洁。

  • 不要用白色勾缝剂，除非你做了长期清洗的打算。

  • 甭管是电视背景墙还是沙发背景，甭管是地砖上墙还是文化砖上墙，一定一定要涂刷背胶或用胶泥来贴，不然很容易空鼓和脱落。

乳胶漆颜色

如果不想要白色乳胶漆，又不知道刷什么颜色好，我给你个建议，暖色调的设计风格，就往白色乳胶漆里调点黄色，刷成不起眼的米白色，冷色调的风格，就兑点黑色，刷成不起眼的浅灰色，以我看过无数套别人家房子的经验来看，墙上刷这两个颜色，属于百搭不会出错的。

乳胶漆品牌

乳胶漆品牌的选择没必要纠结，有些人总觉得乳胶漆无比重要，好像一个选不好就要中毒身亡，纯粹是被忽悠瘸了，现在用的都是水性漆，甲醛释放量在装修材料里连前五都挤不进去，说白了就是个墙面装饰，只要买大厂的，走正规渠道，别去路边小店买假货，基本都不用担心环保问题。

腻子

  • 重要的事情说3遍：不要用腻子粉，请使用成品腻子！不要用腻子粉，请使用成品腻子！不要用腻子粉，请使用成品腻子！成品腻子里的胶水至少是合格产品，不想被腻子粉勾兑的劣质胶水里的甲醛熏瞎眼，就听我的用成品腻子。

  • 腻子要买好一些的，因为墙面出现问题，比如乳胶漆脱皮、返碱、开裂，80%都是腻子质量或者腻子施工没做好造成的，腻子又不贵，买好一点的不吃亏。

推荐品牌：都芳、摩利美涂、蓝星刚玉（成都本地牌子，墙面修补能力很强，墙不平的就用它）、立邦。

乳胶漆滚筒刷

如果你要求比较高，一定要买高档漆，而且是进口不包涂刷的那种，一定要看一下商家有没有配送滚筒刷，如果没配的话，最好自己再掏钱买一个贵一点，越贵的乳胶漆，对滚筒刷头的要求也越高，漆工不可能为了你专门去买个高档滚筒刷，到时候几千的乳胶漆只刷出几百的效果，亏大发了。

一些业主喜欢把生活阳台自带的砖铲了刷乳胶漆，如果不封窗的话完全没必要，因为乳胶漆主要是室外用的，你放到室外风吹日晒，不超过2年就得长霉或者脱皮，防水乳胶漆寿命能长点，但肯定比不上墙砖。

保温层铲不铲？

这点主要是南方地区才会存在的困扰，我的建议是能不动就不动。不为别的，主要是铲保温层贵呀，铲墙的报价一般是11元/㎡，铲保温层则要35元/㎡，而且你铲完肯定还要修复，修复又是35一平，没事铲它干嘛？

当然，有些情况下，保温层是必须铲的。

①保温层只做了一半：你不铲墙不平，或者你可以把整面墙找平也可以，但那样成本比铲保温层更高。

②墙上刚好要做造型：比如要做个吊柜，或者挂什么东西，保温层就必须铲了，因为保温层不是实心的，怕挂不住东西。

③保温层质量太差：南方地区的房子其实用不太上保温层，所有一些开发商就偷工减料，保温层松动、凹凸不平，质量很差，这种也必须铲掉，不然会影响后期腻子和乳胶漆上墙的效果。

除开这些情况，都不用铲保温层，如果工长指这你家完整质量没问题的保温层给你说，不铲墙要开裂、乳胶漆要脱落啥的，不用怀疑，他单纯是想多挣你钱，保温层施工也是有标准的，人家都是挂了网的，正常上腻子刷漆绝对裂不了。

空调管道孔

给空调管道开孔建议趁早，最好在水电改造的时候就开好，因为开孔会把墙和房间弄得很脏，水电改造的时候墙还是毛坯状态，不怕脏。
